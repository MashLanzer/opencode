# 🤖 Discord Bot

| Función | Estado |
|---------|--------|
| /start, /brief, /tasks | ✅ |
| /search | ✅ |
| /join, /leave | ✅ |

## Comandos
- `/start` - Info
- `/brief` - Daily Brief
- `/tasks` - Tareas
- `/search [texto]` - Buscar
- `/join` - Unirse a voice
- `/leave` - Salir de voice

## Running
```bash
export DISCORD_TOKEN="tu_token"
python discord-bot.py
```
